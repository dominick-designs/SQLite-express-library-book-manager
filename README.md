# SQL Library Manager

* Manage library books using SQLite 


## Technologies used

* SQLite3
* Sequelize
* JavaScript
* Node.js
* Express
* Pug
* Middleware including ```body-parser``` and ```cookie-parser```
* ```npx express generator``` to generate express files
* sequelize cli to generate sequelize files
